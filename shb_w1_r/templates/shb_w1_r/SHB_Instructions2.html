Each round will proceed as follows:

  1. For each Observer-Guesser pair the computer will send a "secret number" and a "color number" to the Observer.

    2. The sum of the secret number and the color number generates a "message".

  2. The Guesser will only see the message. Note that the Guesser will never see the secret number or the color number.

  3. The Guesser will then guess the value of the secret number, concluding the round.
